<h3>
  1. App Information
</h3>

<p>
  "muffon" is a free cross-platform music streaming client for desktop, which retrieves audio, video, and metadata from various Internet sources and helps you listen to, discover, and organize music in an advanced way.
</p>

<h3>
  2. Information Collected
</h3>

<h4>
  Types of Personal Information or Data Collected:
</h4>

<p>
  <ul>
    <li>Email</li>
    <li>Country</li>
    <li>City</li>
    <li>Birth date</li>
    <li>Gender</li>
  </ul>
</p>

<h4>
  Anonymous Mode:
</h4>

<p>
  "muffon" offers an anonymous mode that requires no sign-up, and no personal information is gathered in this mode.
</p>

<h4>
  Automated Data Collection:
</h4>

<p>
  The error tracking system, provided by Sentry, may collect device information and error data for improving app stability.
</p>

<h4>
  Cookies and Similar Technologies:
</h4>

<p>
  Local storage is used for data persistence, not for tracking users.
</p>

<h3>
  3. Purpose of Data Collection
</h3>

<p>
  User data is collected for the following purposes:

  <ul>
    <li>Account creation</li>
    <li>Profile page information</li>
  </ul>
</p>

<h3>
  4. Data Sharing
</h3>

<p>
  User data is not shared with any third parties, such as advertisers or analytics services.
</p>

<h3>
  5. User Rights
</h3>

<p>
  <ul>
    <li>Users can access, update, or delete their personal data through the app's settings.</li>
    <li>Users can opt-out of data collection by deleting their account or switching to anonymous mode.</li>
  </ul>
</p>

<h3>
  6. Security Measures
</h3>

<p>
  User data is protected with the following security measures:

  <ul>
    <li>The database is secured with encrypted password.</li>
    <li>The server has a whitelist, ensuring limited access.</li>
  </ul>
</p>

<h3>
  7. Data Retention
</h3>

<p>
  User data is deleted almost instantly upon user request. Users can delete their account by clicking the "Delete account" button in the app's settings, confirming with their password.
</p>
